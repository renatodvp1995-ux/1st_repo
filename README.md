# Math Study 13

Math Study 13 is a beginner-friendly Android math study app built with Kotlin, Jetpack Compose, and Material 3.

Version 1 includes one home screen with:

- The title **Math Study 13**
- The subtitle **Practice math one topic at a time.**
- Four large topic cards:
  - Fractions
  - Algebra
  - Geometry
  - Word Problems
- A simple coming-soon message when each topic is tapped

## How to open the app in Android Studio

1. Open Android Studio.
2. Select **Open**.
3. Choose this project folder.
4. Wait for Gradle sync to finish.

## How to run the app

1. Start an Android emulator or connect an Android device.
2. In Android Studio, choose the **app** run configuration.
3. Click the green **Run** button.
4. The app will open to the Math Study 13 home screen.

## Project structure

- `app/src/main/java/com/example/mathstudy13/MainActivity.kt` contains the Compose home screen code.
- `app/src/main/AndroidManifest.xml` tells Android which activity launches first.
- `app/build.gradle.kts` contains the Android, Kotlin, Compose, and Material 3 setup.
